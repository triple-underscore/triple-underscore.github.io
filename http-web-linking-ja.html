  3. %位置 ~INCBY 1 ◎ Discard the first character.

  4. ~WHILE 無条件：

    1. ~IF［ %位置↗ ~EQ `DQUOTE$P ］ ⇒＃ %位置 ~INCBY 1； ~BREAK【！return output】
    2. ~IF［ %位置↗ ~EQ "`\^c" ］ ⇒ %位置 ~INCBY 1
    3. ~IF［ %位置↗ ~EQ ε ］ ⇒ ~BREAK【！return output】

  5. ~RET %出力 ◎ Return output.
