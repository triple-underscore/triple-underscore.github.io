パスワードを変更するための周知な URL

A Well-Known URL for Changing Passwords

◎要約

この仕様は、 ある`周知な~URL$（ `well-known URL^en, “きちんと知られた~URL” ）を定義する — 各~siteが［ 自身の~change-password~formを~toolにより発見-可能にする ］ために利用できるような。 この単純な~affordanceは、［ 利用者が自身の~passwordを変更する仕方を見出す ］ことを［ ~software用に助ける仕方 ］を供する。 ◎ This specification defines a well-known URL that sites can use to make their change password forms discoverable by tools. This simple affordance provides a way for software to help the user find the way to change their password.

◎非規範的

~client側の~password管理~softwareは、 認証を要求する~web~siteの［ ~security, 利用能 ］どちらも改善することを助ける — それは ⇒＃ 非同一-~site~passwordの再利用を抑制することにより，~securityを改善する。 ~autofill機能性を供することにより，利用能を増強する。 ◎ Client-side password management software helps improve both the security and usability of websites which require authentication. By proposing a well-known URL for changing passwords, this specification enables password managers to help users change their passwords on sites which support it.
