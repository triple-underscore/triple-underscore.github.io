404 — File not found

The requested page is likely to have had its URL changed (especially, the last component of the path). It may be found in index page.

要請されたページは、 URL （特に，パスの最後の成分）が変更された可能性が高いです。 index ページ から見つかるかもしれません。 （このサイト内のページからここに来たのなら、単にそのページ内のリンク URL の誤記です。）

以下に，［ 削除された／URL が変更された／バージョンが更新された ］ページを挙げます（把握している限り。おそらく記載漏れもあります）。

次に挙げる仕様の和訳ページは、過去にはありましたが，現在はありません：

  • CSS Non-element Selectors（ selectors-nonelement-ja.html ）： 原文仕様が退役したので除去しました。
  • CSS2 — § Visual effects（ css22-visufx-ja.html ）： すべての内容が他の CSS モジュールに取って代わられたので除去しました。

次に挙げる仕様の和訳ページの URL は、過去に変更されました：

  • HTML — § Browsing contexts
  • CSS Overscroll Behavior Module Level 1
  • RFC 6265 — HTTP State Management Mechanism
  • IndexedDB Level 2 （ Level 3 に更新しました）
  • SVG 2 — Changes from SVG 1.1
  • CSS Containment Module
  • CSS Scroll Snap Module Level 1
  • CSS Timing Functions Level 1 （仕様名も CSS Easing Functions に改称されました）
  • High Resolution Time Level 2（ Level は無くなりました）
  • Page Visibility Level 2
  • Performance Timeline Level 2
  • User Timing Level 1（ Level 3 に更新しました）
  • Namespaces in XML 1.0/1.1
  • XML Base (Second Edition)
  • Associating Style Sheets with XML documents
  • Feature Policy（仕様名も Permissions Policy に改称されました）
  • Mixed Content Level 2（除去した旧 Level 1 の URL に変更しました／その後， Level は無くなりました）
  • HTTP — Digest Fields

次に挙げる仕様の和訳は、より新たなバージョンの和訳に置換されました：

  • WebIDL Level 1 ： Level 2 に置換しました。その後 Living Standard になりました。
  • IndexedDB Level 2 ： Level 3 に置換しました。
  • Content Security Policy 1.0 ： Level 2 に置換しました。 Level 3 もあります。
  • Page Visibility Level 1 ： Level 2 に置換しました。
  • High Resolution Time Level 1/2： Level 2 に置換しましたが，その後 Level 3 に置換しました。
  • Performance Timeline Level 1 ： Level 2 に置換しました。
  • User Timing Level 1/2 ： Level 3 に置換しました。
  • CSS Writing Modes Level 3 ： Level 4 に置換しました。 Level 3 もあります。
  • CSS Fragmentation Module Level 3 ： Level 4 に置換しました。
  • CSS Values and Units Module Level 3 ： Level 4 に置換しました。
  • CSS Cascading and Inheritance Level 3/4 ： Level 3 は Level 4 に置換しましたが，その後 Level 5 に置換しました。
  • CSS User Interface Module Level 3 ： Level 4 に置換しました。
  • Media Queries Level 4 ： Level 5 に置換しました。
  • RFC 7615 — HTTP Authentication-Info and Proxy-Authentication-Info Response Header Fields ： 改訂下にある HTTP 仕様内に取り込まれました。
  • Mixed Content Level 1 ： Level 2 に置換しました。
