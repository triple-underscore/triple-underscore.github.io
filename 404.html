404 — File not found

The requested page is likely to have had its URL changed (especially, the last component of the path). It may be found in index page.

要請されたページは、 URL （特に，パスの最後の成分）が変更された可能性が高いです。 index ページ から見つかるかもしれません。 （このサイト内のページからここに来たのなら、単にそのページ内のリンク URL の誤記です。）

以下に，［ 削除された／URL が変更された／バージョンが更新された ］ページを挙げます（把握している限り。記載漏れがあるかもしれません）。

次に挙げる仕様の和訳ページは、過去にはありましたが，現在はありません：

  • CSS Non-element Selectors（ selectors-nonelement-ja.html ）： 原文仕様が退役したので、 除去しました。
  • CSS2 — § Visual effects（ css22-visufx-ja.html ）： すべての内容が後発の CSS モジュールに取って代わられたので、 除去しました。
  • Frame Timing： 原文仕様が退役したので、 除去しました。
  • HTTP Authentication-Info and Proxy-Authentication-Info Response Header Fields （ RFC 7615 ）（ http-authentication-info-header-ja.html ）： HTTP 中核仕様（ RFC 9110 ）に取り込まれまたので、 除去しました。
  • Page Visibility Level 2 （ page-visibility-ja.html ）： HTML に統合され， 原文仕様は退役した（ W3C Discontinued Draft になった）ので、 除去しました。
  • HTML の次に挙げる各節：
    • § Browsing contexts（ HTML-browsing-contexts-ja.html ）,
    • § Session history and navigation（ HTML-history-ja.html）,
    • § Browsing the Web（ HTML-navigation-ja.html ）

    原文の大規模な改修に伴い、日本語訳ページをいったん削除しました。 更新を反映する日本語訳は、別 URL にて公開する予定です。 これらのうち一部の内容の日本語訳は、 すでに，目次にあります （タイトルは改称されています）。
