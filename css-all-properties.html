CSS のプロパティ一覧

このページ は、 CSS の各種プロパティを定義する仕様への参照先, それらの和訳への参照先を集約したものです。 （更新： 2026-01-29 （履歴）） （このサイトの CSS 仕様和訳の一覧）

  • プロパティ一覧における参照先は、 次の表に示すような，当のプロパティを定義する各仕様の安定度を表す略称, その色分けで表しています。

    略称／色分けの凡例
    略称 全部的な名前（日本語） 全部的な名前（正式名称）
    ED 編集者草案 Editors' Draft
    FPWD 最初の公な作業草案 First Public Working Draft
    WD 作業草案 Working Draft
    CR 勧告候補スナップショット Candidate Recommendation Snapshot
    CRD 勧告候補草案 Candidate Recommendation Draft
    PR 勧告案 Proposed Recommendation
    REC 勧告 Recommendation
    LS Living Standard（ “生きた標準”, 非 W3C ）
    JA （日本語訳）
  • 各 略称の末尾に付与された数字は、 当のモジュールのレベルを表します。 （ “レベル 1” は CSS2 より前を意味するのではなく， CSS2 には無かった特能を定義する仕様を表していることに注意。 特に， CSS2 には、この数字は付与されない。）

この一覧は， List of CSS properties, both proposed and standard から得られたデータを元にしていますが、 追加しているプロパティもあります （ CSS モジュール以外の仕様（ SVG など）や W3C 以外の仕様に定義されるものもある）。 また、［ 標準でない仕様 （ CSS2 § Aural style sheets, W3C Group Note など） にて定義されるもの／ 旧来の別名として定義されたもの ］への参照は，省略しています （あるいは元々含まれていません）。
