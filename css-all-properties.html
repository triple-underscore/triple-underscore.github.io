CSS のプロパティ一覧

このページ は、 CSS の各種プロパティと，それらを定義する各 仕様, およびその和訳への参照先リンクを集約したものです。 （更新： 2021-10-01 ） （このサイトの CSS 仕様和訳の一覧）

  • プロパティ一覧における参照先は、次の表に示すような，当のプロパティを定義する各 仕様の 安定度を表す略称, その色分けで表しています。

    略称／色分けの凡例
    略称 全部的な名前
    ED 編集者草案（ Editors' Draft ）
    FPWD 最初の公な作業草案（ First Public Working Draft ）
    WD 作業草案（ Working Draft ）
    LC 最終作業草案（ Last Call Working Draft ）
    CR 勧告候補（ Candidate Recommendation ）
    PR 勧告案（ Proposed Recommendation ）
    REC 勧告（ Recommendation ）
    NOTE WG ノート（ Working Group Note ）
    LS Living Standard（ 非 W3C ）
    JA 日本語訳
  • 各 略称の末尾の数字は、当のモジュールのレベルを表します。 （ “レベル 1” は CSS Level 2 より前を意味するのではなく， CSS 2 には無かった特能を定義する仕様を表していることに注意。 特に CSS 2 には、この数字は付与されない。）

この一覧は， List of CSS properties, both proposed and standard から得られたデータを元にしていますが、追加しているプロパティもあります（ CSS モジュール以外の仕様（ SVG など）や W3C 以外の仕様に定義されるものもある）。 また、省略している［ 仕様／その仕様にしか定義されていないプロパティ ］もあります（ CSS2 は別として、何年も更新されておらず，他の仕様に取って代わられた／退役した／およそ実装されそうにないものなど）。
